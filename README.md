ansible-cuda
=======

Ansible role to install the CUDA toolkit as described in the [NVIDIA CUDA Installation Guide](https://docs.nvidia.com/cuda/cuda-installation-guide-linux/index.html#abstract)

Requirements
------------
Ansible >= 2.4
